## 常见控件说明片段

完整组件目录与「点名不重述」原则见 [../pm-component-catalog/reference.md](../pm-component-catalog/reference.md)。

下列为报表常用短片段，复制到查询条件「说明」列，按实际改枚举与范围。

### 区间月份选择器

```
默认值：当月～当月。
空值不参与过滤。
查询逻辑：列表「统计月份」与所选区间有交集即查出。
```

### 数据权限组件

```
多选。
可选范围受当前登录人数据权限控制。
空值 = 不额外限制（仍受数据权限约束）。
```

### 员工组件

```
1. 支持多选。
2. 可输入范围：系统所有人员 / 仅转运人员（按业务选一种写死）。
3. 默认值：空（空 = 不按姓名过滤）。
```

### 岗位组件

```
1. 支持多选。
2. 默认值：空。
3. 可输入范围：系统所有岗位 / 仅转运岗位。
```

### 单选下拉框（全部/是/否类）

```
1. 枚举项：全部、是、否。
2. 默认值：全部。
3. 查询逻辑：选「全部」时不按该条件过滤。
```

### 查询规则（视角切换）

```
1. 枚举项：{组合视角}、{视角 A}、{视角 B}
2. 默认值：{组合视角}
3. 生效查询字段：{字段列表}
   - 组合视角：{实体 A} 或 {实体 B} 任意一方满足即可
   - 视角 A：仅 {实体 A} 字段参与匹配
   - 视角 B：仅 {实体 B} 字段参与匹配
```

         - 是：取 {结果}
         - 否：{结果字段} 为空
2. **优先级 2：{维度名}**
   …
```

1. {条件 1}
2. {条件 2}

```
