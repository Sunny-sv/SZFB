## 配置行命中

**旧（不推荐）**

```
1. 读取该方案启用的区间维度集合 D；
2. 对每个 d ∈ D 取扫描实际值 v(d)；
3. 缺值处理：任一启用维度取不到有效值 → 本方案视为不匹配，继续执行 4.3 下一层方案；
4. 遍历方案下全部配置行 R，找满足 ∀ d∈D，v(d) 落在该行 d 对应区间 [下限, 上限) 的行；
...
7. 命中 1 行：取该行调整，进入 4.5～4.6；
```

**新（推荐）**

**第一步：取维度和扫描实际值**

读维护页勾选的区间维度，按上文「区间维度与扫描取值」表取出每个维度的实际值。

**第二步：检查能否匹配本方案**

- 已勾选的每一个维度，扫描时都必须能取到实际值。
- 任一维度取不到数 → 放弃本方案，去下一层继续找。

> 举例：勾选了票重量 + 票件数，重量能取到、件数取不到 → 本方案直接不适用。

| 命中情况 | 处理方式 |
|---|---|
| 命中 1 行 | 取该行调整比例 → 按岗位取自有/外包 → 算百分比重量 |
| 0 行命中 | 放弃本方案，继续下一层 |
