写新维护页 PRD 时，按页面复杂度选用参照样例。

## 选用指南

| 页面类型 | 参照样例 | 变更记录 Tab | 按钮表列数 |
|---------|---------|-------------|-----------|

## 样例一：业务参数配置（标准简单页）

**定位**：最常见的 PC 维护页 — 查询 + 新增 + 编辑 + 删除 + 导入导出，**无变更记录 Tab**。

**定稿 PRD**：[docs/业务参数配置.md](../../../docs/业务参数配置.md)（项目内）

### 文档要点

| 章节 | 内容 |
|------|------|
| 概述 | V1.0.0，胡绵飞，当天日期，功能点摘要 |
| 要解决的问题 | 参数分散、调整需研发介入 |
| 3.1 产品方案 | 集中维护、在线增删改查、导入导出 |
| 菜单路径 | 转运可视化平台 - 系统管理 - 业务参数配置（待确认） |

### 模块结构

- **查询条件**：参数编码、参数名称、参数类型、所属模块、状态、创建人、创建时间
- **按钮操作**：新增、编辑、删除、导出、下载模版、导入 — **表格仅两列**
- **数据列表**：参数编码、参数名称、参数值、参数类型、所属模块、状态、备注、创建人/时间、修改人/时间

### 关键规则

- 新增/编辑：参数编码唯一；编辑时编码置灰
- 删除：默认删除，确认文案「删除后不可恢复，请确认是否删除！（共 X 条）」
- 导入：字段校验 → 编码唯一性 → 整份成功或整份失败
- 排序：修改时间降序，相同则参数编码升序
- **不写**变更记录相关内容，**不单独说明**无变更记录 Tab

## 样例二：货量重量配置（复杂规则页）

完整 PRD 见：`/Users/humianfei/Downloads/货量重量配置.md`

### Tab 模块索引

| 模块 | 复杂度 | 要点 | 按钮 |
|------|--------|------|------|
| **实时统一维护** | 简单 | 基础查询 + CRUD + 导入导出 | 新增、编辑、批量编辑结束时间、**终止**、导出、下载模版、导入 |
| **实时特殊维护** | 复杂 | 多组合查询、区间类型、优先级 | 同上 |
| **流水线统一/特殊维护** | 继承 | 与实时对应 Tab 一致，仅写差异点 | 继承 + 差异说明 |
| **变更记录 Tab** | 有 | 变更对比弹窗、有效周期 | 导出 |

### 与 Skill 默认的差异

| 项 | 货量重量配置 | Skill 默认 |
|----|------------|-----------|
| 删除操作 | 「**终止**」 | 「**删除**」 |
| 变更记录 | 有，按钮表三列 | 简单页无，按钮表两列 |
| 有效期限 | 15:00:00 ~ 14:59:59 | 仅规则页适用 |
| 唯一性校验 | 分拨/全国纬度 + 有效期交集 | 简单页仅主键唯一 |

## 写新 PRD 时的选用建议

| 新页面特征 | 参照 |
|-----------|------|
| 单表 CRUD，主键唯一，无审计需求 | **业务参数配置** |
| 多维度匹配 + 区间 + 时效 + 审计 | 货量重量配置 |
| 多 Tab 仅枚举/模版不同 | 货量重量配置的继承写法 |
| 规则不可删 | 货量重量配置的终止写法 |
