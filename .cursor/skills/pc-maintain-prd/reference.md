### 新增弹框分区

**简单页**（业务参数配置类）：字段平铺即可，如编码、名称、值、类型、模块、状态、备注。

**复杂页**（规则配置类）：
1. **基础设定** — 主键维度、组织、类型、有效期等
2. **匹配条件配置** — 业务过滤维度
3. **调整比例 / 业务参数** — 页面特有配置区

## 删除按钮（默认）

1. **操作限制**：选中条数、可删状态
2. **确认弹框**：「删除后不可恢复，请确认是否删除！（共 X 条）」
3. **按钮**：取消关闭；确定执行删除
4. **确定后**：数据移除或逻辑删除，列表刷新
5. 有变更记录 Tab 时：第三列写修改类型 = 删除

---

## 简单页 · 唯一性校验（业务参数配置类）

适用于编码/主键唯一的标准维护页，无需分拨纬度、有效期限：

1. 新增/编辑确定：校验**主键编码**是否已存在 → 提示「{编码字段}已存在，请修改后重试」
2. 导入：文件内主键重复 → 整份失败；与已维护数据冲突 → 整份失败，标注行级错误

---

## 复杂页 · 唯一性 / 重复校验

适用于规则类维护页（分拨纬度 / 全国纬度分开写）：

1. 按 {匹配维度} 校验已维护数据是否有交集或包含关系
2. 有交集 → 再校验有效期限是否有交集
3. 仍有交集 → 展示「数据重复提示弹框」
   - 取消：关闭弹框，不执行操作
   - 确认：关闭所有弹框，自动查询出重复数据

---

## 终止按钮（特殊页面）

仅用于数据不可物理删除的规则/配置页（参考货量重量配置）。

1. **操作限制**：通常仅生效中/待生效可终止
2. **确认弹框**：「终止后不可撤销，请确认是否终止！（共 X 条）」
3. **确定后**：结束时间改为当前时间、状态改为终止；若有优先级则写重排规则
4. 有变更记录 Tab 时：第三列写修改类型 = 终止

### 状态判定（有时效的规则页）

| 状态 | 判定规则 |
|------|---------|
| 生效中 | 有效期限包含当前时间 |
| 待生效 | 开始时间 > 当前时间 |
| 已失效 | 结束时间 < 当前时间 |
| 终止 | 手动终止（仅不可删数据的特殊页） |

### 有效期限惯例（规则页）

- 开始时间：日期 + 固定 `15:00:00`
- 结束时间：日期 + 固定 `14:59:59`，必须大于开始时间
