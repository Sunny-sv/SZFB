### V1.1.0

#### 5.3.0 三报表共性改动

##### 货量类字段口径变更（全部 Tab）

**识别规则：**列表中所有原按**操作结算重量**统计的货量类字段（不含本次新增的「原始操作结算重量」）。

| 变更项 | 变更前 | 变更后 |
|---|---|---|
| 取值口径 | 操作结算重量 | 百分比重量 |
| 字段名 | 不变 | 不变 |
| 展示格式 / 空值展示 | — | 沿用原规则 |

各报表具体字段见 5.3.1～5.3.3；非数据汇总 Tab 仅改口径，不增删列。

#### 新增字段「原始操作结算重量」（仅数据汇总 Tab）

| 项 | 说明 |
|---|---|
| 汇总维度 | **省区、分拨**（数据汇总 Tab 内切换；**两个维度均**新增「原始操作结算重量」） |
| 列位置 | 各维度列表均在对应锚点列右侧（锚点列见 5.3.1～5.3.3，两个维度锚点列名相同） |
| 取值口径 | 按操作结算重量汇总，取当前汇总行维度下的**总重**（一列总数，不拆分子项） |
| 可见性 | 仅总部账号可见；非总部列表不展示 |

##### 导出

| 项 | 规则 |
|---|---|
| 货量类字段 | 与列表一致，按百分比重量 |
| 原始操作结算重量 | 仅数据汇总 Tab；**省区、分拨**两个维度均适用；总部导出含该列，非总部不含 |
| 导出文件名 | 不变 |
| 总部判定 | 待讨论 |
