# 二、要解决的问题

|**版本**|**问题**|
|---|---|
|V1\.0\.0|1. 特定时间段离职、复职的员工，存在反复离职复职现象，影响分拨作业效率。<br>2. 此类复职员工无差异化薪资管控，与正常入职员工薪资无区分，无法约束其流动，也影响在职员工公平感。<br>3. 复职员工春节前后返岗积极性不足，可能导致分拨人员短缺，影响节日业务开展。<br>4. 针对该类员工的复职管控方案不明确，流程、薪资、激励规则模糊，增加管理难度。|

# 四、需求分析

## 4\.1 需求价值

|**版本**|**需求价值**|
|---|---|
|V1\.0\.0|1. 运营稳定：约束人员反复流动，保障分拨作业连续性，提升作业效率。<br>2. 管理规范：明确复职薪资、激励规则，填补管理空白，降低管理成本。<br>3. 成本优化：通过差异化薪资控制用工成本，借助返岗奖励减少临时用工需求。<br>4. 人员激励：平衡在职员工公平感，提升复职员工返岗意愿及留存率。<br>5. 长效支撑：形成标准化管控机制，助力建立稳定人员队伍，支撑业务发展。|
